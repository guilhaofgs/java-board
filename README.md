# Board de Tarefas em Java

Este projeto implementa um board de tarefas utilizando conceitos de **Programação Orientada a Objetos (POO)** em Java.

## Funcionalidades
- Adicionar tarefas
- Listar tarefas
- Concluir tarefas

## Como executar
1. Salve os arquivos na pasta `src/`
2. Compile todos os arquivos Java:
```bash
javac src/*.java
```
3. Execute:
```bash
java -cp src Main
```
